Política de Privacidade - Água Já

Última Atualização: 14/04/2025

1. Introdução

O aplicativo "Água Já" tem como objetivo ajudar os usuários a manter uma hidratação saudável, permitindo o registro do consumo de água e o envio de lembretes para beber água. Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações do usuário e esclarece que nenhum dado pessoal é enviado a servidores remotos.

2. Dados Coletados

Dados Localmente Armazenados: O aplicativo registra o horário em que o usuário clica no botão "Bebi água". Essas informações são armazenadas localmente no dispositivo utilizando AsyncStorage (ou mecanismo similar) e não são compartilhadas ou enviadas a terceiros.

Informações Pessoais: O app não solicita, coleta ou processa nenhuma informação pessoal, tais como nome, endereço, e-mail ou localização.

3. Uso dos Dados

As informações de histórico de hidratação são utilizadas exclusivamente para que o usuário acompanhe seu consumo de água. Os dados armazenados localmente não são compartilhados com nenhum serviço externo e permanecem no dispositivo do usuário.

4. Compartilhamento de Dados

O aplicativo não compartilha, vende, ou divulga informações dos usuários a terceiros. Todas as informações ficam armazenadas de forma segura e localmente no dispositivo do usuário.

5. Privacidade das Crianças

O app foi desenvolvido para todos os públicos e, conforme as diretrizes, não coleta dados de crianças menores de 13 anos. Caso o aplicativo venha a ser direcionado a esse público, tomamos as medidas necessárias para cumprir a legislação aplicável, respeitando a privacidade dos menores.

6. Segurança dos Dados

Implementamos práticas técnicas que visam proteger as informações armazenadas localmente no dispositivo. Recomendamos que o usuário mantenha seu sistema operacional e o aplicativo atualizados para garantir o melhor nível de segurança.

7. Alterações nesta Política de Privacidade

Podemos atualizar esta Política de Privacidade periodicamente para refletir alterações em nossas práticas ou devido a mudanças na legislação. Recomendamos que os usuários revisem esta política regularmente para se manterem informados sobre como protegemos as informações.
